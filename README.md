# Experiment Documentation

This repository contains the prototypical implementation used to generate results presented in our publication "Exploring the Cost Landscape of Variational Quantum Algorithms".

It involves the evaluation of metrics including:

- Total Variation
- Fourier Density
- Inverse Gradient Standard Deviation
- Scalar Curvature

Python version: 3.10.9

``pip install -r requirements.txt``
